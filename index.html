모든 기능

복잡한 세팅 없이 처음 실행하면 바로 사용할 수 있습니다.

🏨

실시간 객실 현황판

이용가능·사용중·청소중·점검중·고장 상태를 색상으로 한눈에 확인. 전체 통계와 오늘 매출을 상단에 항상 표시합니다.

✅

체크인 / 체크아웃

고객명·전화번호·요금·예상 체크아웃 시각 입력. 체크아웃 시 자동으로 이력을 저장하고 객실을 초기화합니다.

❌

노쇼 처리 & 환불

예약 부도(노쇼) 시 위약금 금액을 입력해 별도 처리. 부분 환불도 지원하며 초과 환불은 자동 방지됩니다.

📊

일마감 정산 & CSV

날짜별 체크인·체크아웃 수, 총 매출·환불·순매출 자동 집계. CSV 파일로 내보내기해 엑셀에서 바로 확인 가능.

👥

역할별 권한 관리

관리자(Admin)·프런트(Counter)·청소(Cleaning) 3단계 권한 분리. 청소 직원은 상태 변경만, 프런트는 체크인·정산까지 가능.

🔒

개인정보 보호

전화번호 자동 마스킹(010-****-5678). 보관 기간(기본 365일) 초과 이력은 [익명] 처리로 개인정보보호법 대응.

🔑

PIN 인증 & 접근 로그

SHA-256 암호화 PIN으로 안전한 로그인. 모든 작업(체크인·정산·권한 변경)이 시간·사용자와 함께 자동 기록됩니다.

💾

완전 오프라인 작동

인터넷 없이도 100% 작동. 데이터는 내 PC에만 저장(SQLite). 업데이트도 선택 사항으로 자동 설치 없음.

FAQ

해와달 자주 묻는 질문

해와달 PMS 설치나 가입이 필요한가요?
가입은 필요 없습니다. setup.exe를 실행해 설치한 뒤 바탕화면 아이콘으로 바로 사용하시면 됩니다. Windows Defender 경고가 뜨면 "추가 정보 → 실행"을 선택하시면 됩니다.
인터넷이 없어도 작동하나요?
네, 완전히 오프라인에서 작동합니다. 데이터는 프로그램과 같은 PC의 AppData 폴더에 저장되며, 외부 서버에 전송되지 않습니다.
객실 수를 변경할 수 있나요?
기본 설정은 1~3층 각 10호실(총 30실)입니다. 객실 수가 다른 경우 문의주시면 맞춤 설정으로 제공해 드립니다.
직원이 여러 명인데 같이 쓸 수 있나요?
관리자·프런트·청소 3단계 계정을 생성해 역할별로 사용할 수 있습니다. 단, 현재 버전은 한 PC에서 사용하는 단일 매장 용도입니다. 네트워크 다중 접속이 필요하시면 별도 문의해 주세요.
데이터는 어디에 저장되나요?
C:\Users\사용자명\AppData\Roaming\HaeWaDalPMS\motel.db 파일에 저장됩니다. 이 파일을 복사해두면 백업이 됩니다.
다른 PC/모바일과 연동할 수 있나요?
네. 다운로드 섹션의 Supabase 연동 스타터를 사용하면 여러 기기에서 같은 데이터 기준으로 운영할 수 있습니다. 초기 설정 후 메인 PC에서 자동 동기화로 운영합니다.
스타터 배치파일을 눌렀는데 바로 종료돼요
최신 스타터 ZIP을 다시 받아 압축 해제 후 SUPABASE_STARTER_README.txt를 먼저 확인해 주세요. 실행 중 오류가 있으면 창이 유지되도록 수정된 버전(2026-03-08)입니다.
홈페이지 다운로드가 안 열리면 어떻게 하나요?
다운로드 섹션의 첫 번째 버튼이 실패하면 두 번째 버튼 구글드라이브 대체 다운로드를 눌러 설치파일을 내려받아 주세요.
최신 배포가 반영됐는지 어떻게 확인하나요?
2026-03-11 배포분부터 새 달/건물 로고 아이콘이 표시됩니다. 아이콘이 예전이면 브라우저 새로고침(Ctrl+F5) 후 다시 다운로드해 주세요.
배포·재판매가 가능한가요?
아니요. 본 프로그램은 계약된 사업장 전용 라이선스이며, 무단 복제·재배포·재판매는 금지됩니다. 지점 추가 사용은 별도 계약이 필요합니다.
